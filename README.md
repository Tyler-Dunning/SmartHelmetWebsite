# Smart Helmet Web Interface
*** Early Work In Progress ***

Smart Helmet is a combat helmet with hardware components that track and detect impact data.
It's goals are to reduce the risk of serious head injuries and assist judges in scoring martial arts competitions.

This website will be responsible for displaying all of the data gathered by the Smart Helmet to the user.

* This repository contains the frontend for the Smart Helmet Application
* The frontend links to a Google Cloud Platform backend project that handles:
    - Web Server API requests
    - Database storage
    - User Authentication
    - POST requests from the Smart Helmet Raspberry Pi
